About

With the combination of JavaScript, HTML and CSS, the project explores core visual computing techniques through the design and implementation of interactive, browser-based graphics systems.

It focuses on transforming visual data into structured, real-time representations using JavaScript and HTML.

The work demonstrates strong fundamentals in graphics pipelines, rendering logic, and interactive system design.

Applied and optimize Minecraft texture assets using Aseprite to ensure an accurate rendering and visual fidelity in a HTML web browser.
