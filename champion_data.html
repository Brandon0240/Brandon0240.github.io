About

This project analyzes competitive League of Legends gameplay data collected directly from Riot Games’ developer API to uncover statistics not publicly available on popular analytics platforms.

Using large-scale match data, we engineered a custom data-gathering, cleaning, and analysis pipeline to evaluate how factors such as first blood, item builds, champion roles, and item combinations impact win rates.

The results reveal statistically significant patterns such as high-impact item–champion synergies and role-dependent scaling.

It provides players with deeper, data-driven insights into optimal gameplay decisions.

Data Sorting

• Utilize PyCharm's debugger to sort through a medium data set, which was 2.4 GB and contained over 460490 rows.

• Used list appending, panda data frame sorting, exploding and other functions to transform data into easier to handle numbers.

• The sorted gameplay data was analyzed and visualized using bubble charts to identify optimal item builds for League of Legends champions.

• This is shown through emphasizing the correlations between item usage frequency and win rate.
